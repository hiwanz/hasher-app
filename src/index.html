Hasher

Cryptographic hashing and data conversion utilities. All calculations run locally — your data never leaves this device.

Features

  • Hash: MD5, SHA-1, SHA-2 (224/256/384/512), RIPEMD-160, MD4, Whirlpool
  • HMAC: MD5, SHA-1, SHA-2 (224/256/384/512), RIPEMD-160, MD4
  • CRC: CRC-8, CRC-16, FCS-16, FCS/CRC-32
  • Cipher (OpenSSL compatible): AES-256, DES, 3DES, Rabbit, RC4, RC4Drop
  • Network: Subnet calculator, IP ↔ Dec/Bin/Hex
  • Time: Unix ↔ Datetime, RFC-1123, ISO 8601
  • Numbers: Dec ↔ Hex / Bin / Roman
  • Strings: ASCII, UTF-8, UTF-16 ↔ Hex
  • Encode: Base64, ROT-13, URI, HTML
  • Morse: Text to Morse code

Usage

Type in the input field and click any result to copy it to the clipboard. Password field appears automatically for HMAC and Cipher modes.
